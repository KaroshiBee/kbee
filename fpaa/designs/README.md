# Designs Layout

This directory holds the active kbee FPAA designs.

## Active AD2 Designs

- `kbee-04.ad2` — canonical four-chip export (residue + NOR/XOR/AND accumulators)
- `kbee-05.ad2` — single-chip hardware-export copy of `kbee-04`, constrained to
  the four CABs of one AN231E04 for on-board deployment

## Active Reports

Under `fpaa/designs/reports/`:

- `kbee-04/` — HTML export and companion PNGs for the active design
